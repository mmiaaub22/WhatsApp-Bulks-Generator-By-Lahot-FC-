Generator Panel

Generate realistic US-style phone numbers in the format +1 (NPA) NXX-XXXX with random first/last names and export everything as a CSV file.

Pick from the list or choose "Custom" and type any valid US area code.

Minimum 50, maximum 15,000 per batch.

We auto-generate valid-style prefixes for the NANP format (2–9XX; no N11, no 555). Uncheck any you don’t want to use. Or set a single custom prefix below.

If set, every number will use this exact prefix (must be 3 digits like 204; first digit 2–9, not N11, not 555).
